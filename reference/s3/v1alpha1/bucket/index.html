Metadata#

PropertyValue
ScopeNamespaced
KindBucket
ListKindBucketList
Pluralbuckets
Singularbucket

In terms of implementation, a Bucket is a resource. An Amazon S3 bucket name is globally unique, and the namespace is shared by all Amazon Web Services accounts.
